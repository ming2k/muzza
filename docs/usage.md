# Usage Guide

## Basic Operations
* **Import Media**: Click the `+` tile in the Media Bin to open the in-app browser and select files.
* **Insert to Timeline**: Drag media from the Media Bin onto a Timeline track. The system automatically splits video and audio.
* **Move Clips**: Left-click and drag any clip body on the timeline to move it across tracks or time.
* **Trim Clips**: Hover over the left or right edge of a clip (in Select mode) and drag to adjust the in/out points.
* **Split Clips**: Switch to Razor mode (`C` key or `CUT` button) and click anywhere on a clip to split it into two adjacent clips.
* **Delete Clips**: Select a clip and press `Delete` to remove it from the timeline.
* **Adjust Track Height**: Drag the bottom edge of a track to resize it. Audio waveforms will scale automatically.
* **Scrub Playhead**: Left-click or drag on the timeline ruler or empty space to move the playhead.
* **Timeline Zoom**: Use `Ctrl` + `Mouse Wheel` or the `=` and `-` keys to zoom in and out of the timeline. The zoom is centered on your mouse position.
* **Timeline Scroll**: Use the `Mouse Wheel` to scroll the timeline horizontally.

## Shortcuts
| Key | Action |
|---|---|
| `Space` | Toggle Play/Pause |
| `Ctrl` + `Wheel` | Zoom Timeline |
| `=` / `-` | Zoom In / Out |
| `Wheel` | Scroll Timeline Horizontally |
| `V` | Switch to Select Tool |
| `C` | Switch to Razor Tool |
| `Delete` | Remove Selected Clip |
| `Esc` | Quit Application |
| `Left Click` | Select Clip / Scrub Timeline / Split Clip (Razor mode) |
| `Right Click` | Open Media context menu (Insert/Delete) |

## Preview Modes
* **Source Preview**: Select a media item in the Media Bin to preview the original file in the monitor.
* **Timeline Preview**: The monitor automatically switches to composite mode when the playhead is over active clips.
